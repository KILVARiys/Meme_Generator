# Meme_Generator
# 🎭 Генератор мемов

Веб-приложение для создания мемов с возможностью загрузки собственных изображений и наложения текста.

## ✨ Возможности

- 📸 Загрузка собственных изображений (PNG, JPG, JPEG, GIF, BMP)
- 📝 Наложение верхнего и нижнего текста
- 🎨 Автоматическое центрирование текста
- 🔍 Предпросмотр изображения перед загрузкой
- 💾 Скачивание готового мема в формате JPEG
- 🎯 Поддержка изображений с прозрачностью (автоконвертация в RGB)
- 📱 Адаптивный дизайн

## 🛠 Технологии

- **Backend**: Python, Flask
- **Обработка изображений**: Pillow (PIL)
- **Frontend**: HTML5, CSS3, JavaScript
- **Шрифты**: Arial (с fallback на системный)

## 📋 Требования

- Python 3.7+
- pip (менеджер пакетов Python)

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/meme-generator.git
cd meme-generator
```

### 2. Установка зависимостей

```bash
pip install flask pillow
```

### 3. Подготовка шрифта

Скачайте файл шрифта Arial (.ttf) и поместите его в корневую папку проекта, или используйте системный шрифт, изменив путь в коде.

### 4. Запуск приложения

```bash
python app.py
```

### 5. Открытие в браузере

Перейдите по адресу: `http://127.0.0.1:5000`

## 🔧 Использование

1. **Загрузите изображение**
   - Нажмите кнопку "Выберите файл"
   - Выберите изображение с вашего компьютера
   - Сразу увидите предпросмотр

2. **Добавьте текст**
   - Введите текст для верхней части мема
   - Введите текст для нижней части мема

3. **Создайте мем**
   - Нажмите кнопку "Создать мем"
   - Готовое изображение автоматически скачается

## ⚙️ API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/` | Главная страница |
| POST | `/upload` | Загрузка изображения и создание мема |
| GET | `/download/<filename>` | Скачивание готового мема |

## 🔒 Безопасность

- ✅ Проверка типа загружаемых файлов (только изображения)
- ✅ Ограничение размера файла (16 MB)
- ✅ Уникальные имена файлов (UUID)
- ✅ Автоматическая очистка временных файлов
- ✅ Конвертация RGBA в RGB для совместимости с JPEG
